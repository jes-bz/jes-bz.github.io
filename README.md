## Scripts

This project includes several scripts to help with development and optimization:

- `scripts/compress_videos.sh` - Compresses video files to reduce their size
- `scripts/optimize_images.sh` - Optimizes images to reduce their file size

These scripts require `ffmpeg` and `ImageMagick` to be installed on your system.
